== Description ==
Improve your website's performance and speed by removing emoji scripts and styles if you don't need them.

If you don't need or want Emoji, then it is better to remove/dequeue Emoji scripts and styles for better performance. Less JS and CSS = a good thing!

*   Before enabling the plugin, first check your page source for "Emoji". You'll find scripts and styles inside <hea> section.
*   After enabling the plugin, check for "Emoji", you won't find anything.

For more information, Please visit [Remove Emoji Styles & Scripts](https://www.inthiscode.com/remove-emoji-styles-scripts).

== Installation ==
Install using Plugins->Add New or manually upload using WordPress plugin uploader.
Activate the plugin, and go to the Settings -> Remove Emoji Styles & Scripts.
Check the "**Enable Plugin**" checkbox to remove emoji scripts and styles.
